# 条件
- 自作譜面をつかうこと
- speed譜面であること (休憩地帯やアクセントで他傾向はOK)

この2つ守ってくれるならどんなクソ譜面でも受け付けます

# コンバート方法
1. https://github.com/kovaxis/osu2sm/releases/tag/v0.1 ここからosu2sm.exeをダウンロード
2. ターミナルが開くのでそこにosuのsongsフォルダをD&D (フォルダの中にある曲がすべて変換されてしまうので別フォルダを作成しておくと良い)
3. このpackのフォルダをD&D
4. 作成されたフォルダの中にcdtitle.jpg (マッパーのアイコン), 曲のバナー, .smファイルを編集する

バナー, アイコンは自作してね
